## 📋 專案簡介

本專案是廖天佑（Bless Liao）的個人品牌網站，採用**AI搜尋優化（GEO）**最佳實踐設計，展示其在**生成式引擎優化**、**數位行銷策略**和**內容架構設計**方面的專業成就。網站完全基於廖天佑的真實專業背景和專案經歷製作，旨在成為AI時代個人品牌網站的標竿範例。

### 🌟 網站亮點

- **🤖 AI搜尋優化**：完整實施GEO策略，針對ChatGPT、Google SGE、Perplexity等AI平台優化
- **📊 權威內容展示**：整合AIOGEO知識庫、MetaAd廣告投手養成中心等重要專案
- **🌐 雙語並行架構**：中英文內容並存，服務國際與本土受眾
- **📱 響應式設計**：完美適配桌面、平板、手機等各種裝置
- **⚡ 高性能優化**：快速載入、SEO友好、現代化視覺體驗

## 📈 部署指南

### 🌐 GitHub Pages部署

#### 方法一：直接上傳

1. **創建儲存庫**
   
   ```
   - 登入GitHub
   - 點擊 "New repository"
   - 儲存庫名稱：blessliao-profile
   - 設定為Public
   ```
1. **上傳檔案**
   
   ```
   - 點擊 "uploading an existing file"
   - 拖拽index.html到頁面
   - Commit changes
   ```
1. **啟用GitHub Pages**
   
   ```
   - Settings → Pages
   - Source: Deploy from a branch
   - Branch: main, Folder: / (root)
   - Save
   ```

#### 方法二：Git指令部署

```bash
# 1. 初始化Git儲存庫
git init

# 2. 新增檔案
git add .

# 3. 提交變更
git commit -m "Initial commit: Bless Liao personal website"

# 4. 新增遠端儲存庫
git remote add origin https://github.com/你的用戶名/blessliao-profile.git

# 5. 推送到GitHub
git push -u origin main
```

### 🔧 自訂網域設定

1. **購買網域**：建議 `blessliao.com` 或 `bless-liao.com`
1. **DNS設定**：
   
   ```
   A記錄：185.199.108.153
   A記錄：185.199.109.153
   A記錄：185.199.110.153
   A記錄：185.199.111.153
   CNAME：www → 你的用戶名.github.io
   ```
1. **GitHub設定**：
- Settings → Pages → Custom domain
- 輸入網域名稱並儲存
- 等待DNS驗證完成
